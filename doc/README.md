## Guide d'Installation et de Lancement
### Prérequis
- Java 11
- Maven 3.6

### Étapes d'Installation
```bash
git clone https://github.com/SI4-Equipe-G/projet.git
cd projet
mvn clean install
```

### Lancement de l'Application
```bash
java -jar target/projet.jar
```
